Dinámicas de juego
Diseño de bucles de juego y mecánicas de juego desde la creación de prototipos hasta la ejecución. Desarrollo de comportamientos de IA para personajes y sistemas utilizando FSM's. Diseño de patrones para múltiples sistemas de juego orientados a las motivaciones del jugador.

Portafolio

That Freezing Night

Juego de aventuras en primera persona. Trabajé con Finite State Machines para la IA del White Warrior y un sistema de pistas para encontrar objetos, también configuré la iluminación general, la interfaz de usuario y el posprocesamiento. Se puede completar con dos finales diferentes.

  • Check it out!

Unknown Trench

Walking Simulator en primera persona donde controlas a un buzo de rescate que busca a su compañero de tripulación perdido. Trabajé en las secuencias de juego usando Sequencer y también diseñé la persecución del jefe monstruo y el level streaming.

  • Check it out!

Letters from the Outside

Juego de rompecabezas donde se envían cartas. Trabajé en un sistema de traducción y una interfaz de arrastrar y soltar que imita una aplicación de escritorio. Hay un temporizador para resolver acertijos y cada letra se cambia día a día.

  • Check it out!

The Hustler Program

Juego runner con un medidor de gas que se agota cada vez que el jugador se propulsa. Tiene un sistema para generar nuevas filas de monedas y enemigos y también IA de patrulla básica para enemigos.

  • Check it out!
